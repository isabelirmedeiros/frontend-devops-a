# Git Comandos 

- git clone = clona um repositório github para pc
- git status = mostra o status do repositório local
- git add nome-arquivo = adiciona no stage changes o arquivo
- git add . = adiciona todos os arquivos no stage changes
- git checkout -b = cria uma nova brand
- git commit -m = salva os arquivos no repositório local 
- git push = envia o repositório
